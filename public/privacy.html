Privacy Policy

Last reviewed: May 25, 2026

This policy describes BuildOrbit's expected production data handling. It should be reviewed by counsel before public launch or material data-processing changes.

Data We Process

BuildOrbit may process account email addresses, authentication sessions, API tokens, GitHub OAuth connection metadata, prompts, generated artifacts, deployment metadata, billing status, logs, and provider event records.

Service Providers

BuildOrbit uses infrastructure and service providers for hosting, PostgreSQL, object storage, email delivery, AI generation, GitHub OAuth/API access, and Stripe billing.

Data Retention

Production artifacts and run history are retained while the account remains active unless deleted through product workflows or support. Security logs, provider events, and billing records may be retained as needed for fraud prevention, audit, tax, dispute, and legal obligations. Authentication tokens are stored hashed or encrypted where applicable and expire or can be revoked.

Secrets

Secrets are stored in platform environment configuration, not in the repository. Users should not paste credentials into prompts or generated code.

Email

Transactional email includes magic links, password resets, welcome messages, credit warnings, and subscription confirmations. Delivery events may be logged for operational troubleshooting.

User Choices

Users can disconnect GitHub, manage billing through Stripe, revoke API tokens, and request account or data deletion through support.

Contact

Privacy questions should be sent through the support channel listed in the application.
