Live Pipeline View During Build

When a build is running, you see it live. Each phase lights up as it executes. Agent reasoning transcript scrolls below in real time. You can watch every decision as it happens — or come back later and read the full log.

  • Phase cards — Intent Gate, Plan, Scaffold, Code, Save, Verify with live status (pending/running/completed/failed)
  • Agent Reasoning panel — Full transcript per phase: what the agent decided, why, and what it considered but rejected
  • Cost tracker — Running cost estimate updates in real time
  • Preview pane — Live preview of generated output as files are created

Run Detail Post Build

After a build completes (or fails), the run detail page gives you the full picture.

  • Phase-by-phase breakdown — Each phase's output, timing, and cost
  • Agent reasoning timeline — Every decision the agent made, with context and justification
  • Audit report — VERIFY phase output: critical and advisory findings, with file/line references
  • File tree viewer — Browse the generated output, inspect any file
  • Deploy link — Live URL, PR link, or download, depending on deployment type
  • Source + prompt — The original task description and source context

History & Export

All past runs are stored and searchable. You can filter by status, date, source repo, or outcome. Export any run's full transcript and audit report as a standalone document — useful for compliance logging or client handoff.
